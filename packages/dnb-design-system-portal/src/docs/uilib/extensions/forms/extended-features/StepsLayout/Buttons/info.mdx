## Description

`StepsLayout.Buttons` Is a composition for a row containing both PreviousButton and NextButton.

```jsx
import { StepsLayout, Form } from '@dnb/eufemia/extensions/forms'

render(
  <Form.Handler>
    <StepsLayout>
      <StepsLayout.Step title="Step 1">
        <StepsLayout.Buttons />
      </StepsLayout.Step>
    </StepsLayout>
  </Form.Handler>,
)
```
